## Como executar?
Baixe o projeto
```
 git clone https://github.com/adrianotorres/k121
 cd k121
```
### Executando com Docker
Certifique-se de ter o docker e docker-compose instalados
```
 docker-compose build
 docker-compose up
```

### Executando com NPM local
```
 npm run build
 npm run back:dev
```
ou, em duas abas separadas
```
 npm run front:dev
 npm run back:dev
```

### Testes
```
 npm run test
```
